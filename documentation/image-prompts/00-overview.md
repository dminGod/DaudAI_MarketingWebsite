# Image Generation Overview for DaudAI Marketing Website

## Brand Guidelines for All Images

### Color Palette
- **Primary Brown**: #5C3D2E (dark chocolate brown)
- **Primary Orange**: #E07A00, #F5921B (vibrant orange)
- **Light Orange**: #FFB347 (warm amber)
- **Cream**: #FFF8F0 (warm white)
- **White**: #FFFFFF

### Style Direction
- Modern, clean, tech-forward
- Warm and approachable (matching the energetic mascot)
- Professional but not corporate
- Consistent with the existing mascot character (anime-style boy with glasses)

### Reference: Existing Mascot
The mascot is an anime-style young boy with:
- Brown messy hair
- Glasses
- Orange/red sleeveless shirt
- Dynamic action pose (running/punching forward)
- Energetic expression
- Surrounded by dynamic orange/brown debris effects

## Output Specifications

After generation, images should be placed in:
```
c:\code\daudai\marketing-website\public\images\
```

File naming convention:
- `og-image.png` - Social share image
- `screenshot-android.png` - Android mockup
- `screenshot-ios.png` - iOS mockup
- `hero-bg.png` - Hero background (if used)
- `security-illustration.png` - Security visual
- `step-1-desktop.png` - How it works step 1
- `step-2-relay.png` - How it works step 2
- `step-3-mobile.png` - How it works step 3

# Hero Background (Optional)

## Specifications
- **Ratio**: 16:9 (landscape, wide)
- **Purpose**: Subtle background for the hero section (used with low opacity)
- **Output File**: `hero-bg.png`

## Prompt

```
Create an abstract tech background illustration for a developer tools website.

ELEMENTS:
- Flowing, curved lines suggesting connectivity and data flow
- Abstract geometric shapes (circles, rounded rectangles)
- Subtle code/terminal window outlines
- Device silhouettes (laptop, phone) very faintly in background
- Particle effects and small dots
- Gradient mesh areas

STYLE:
- Abstract and minimal
- Warm color palette only:
  - Orange tones: #E07A00, #F5921B, #FFB347
  - Brown tones: #5C3D2E, #7A5240
  - Cream: #FFF8F0
- Very soft, low contrast (this will be used as subtle background)
- No text
- Ethereal, flowing feel
- Tech-forward but warm

IMPORTANT:
- Keep it subtle - this goes BEHIND content
- Use transparency/fading at edges
- Center should be lightest (content goes there)
- Avoid harsh contrasts

Landscape orientation, 16:9 aspect ratio
```

# OG Image (Social Share Image)

## Specifications
- **Ratio**: 1.91:1 (landscape)
- **Recommended Size**: 1200x630 pixels
- **Purpose**: Displayed when sharing links on Twitter, Facebook, LinkedIn, Discord, Slack, etc.
- **Output File**: `og-image.png`

## Prompt

```
Create a professional marketing banner for "DaudAI" - a mobile app that lets developers control their AI coding sessions remotely.

COMPOSITION:
- Left side: The DaudAI logo/text "daud.ai" in stylized brown and orange lettering
- Center/Right: A visual showing a smartphone connecting to a desktop computer, with glowing connection lines between them
- Include subtle code/terminal elements in the background
- The mascot character (anime boy with glasses, orange shirt) can appear small in corner giving thumbs up

STYLE:
- Modern tech product aesthetic
- Clean and professional
- Warm color palette: brown (#5C3D2E), orange (#E07A00, #F5921B), cream (#FFF8F0)
- Gradient background from cream to light orange
- Subtle glow effects on connection lines

TEXT TO INCLUDE:
- "daud.ai" as the main brand
- Tagline: "Control Claude Code from Anywhere"

MOOD:
- Professional yet approachable
- Tech-forward
- Trustworthy
- Energetic

Landscape orientation, 1.91:1 aspect ratio
```
